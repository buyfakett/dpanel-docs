# dpanel

DPanel 是一款 Docker 可视化管理面板，提供了完善的容器、镜像、存储、网络等功能。

# 功能介绍

- 全中文的界面，相比于 portainer 更友好
- 完善的容器管理功能，提供域名转发，文件管理，日志监控等功能。轻松对容器内的文件进行管理。
- 提供容器之间的关联功能，便于多个容器之间的互相依赖访问。
- 提供多种环境的基础镜像和模板，可以快速构建属于自己的镜像。
- 可以通过 Zip 或是 Git 构建镜像，快速实现可持续化构建。
